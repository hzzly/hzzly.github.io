1.创建 String 对象的语法：

2.属性

属性 描述
constructor 对创建该对象的函数的引用
length 字符串的长度
prototype 允许向对象添加属性和方法

3.常用方法

方法 描述
charAt() 返回在指定位置的字符
concat() 连接字符串
indexOf() 检索字符串
lastIndexOf() 从后向前搜索字符串
match() 找到一个或多个正则表达式的匹配
slice() 提取字符串的片断，并在新的字符串中返回被提取的部分
split() 把字符串分割为字符串数组
toLowerCase() 把字符串转换为小写
toUpperCase() 把字符串转换为大写
toString() 返回字符串
valueOf() 返回某个字符串对象的原始值
