四、题外话

基本类型值与引用类型值具有如下特点：

  • 基本类型值在内存中占据固定大小的空间，因此被保存在栈内存中
  • 从一个变量向另一个变量复制基本类型的值，会创建这个值得一个副本
  • 引用类型的值是对象，保存在堆内存中
  • 包含引用类型值得变量实际上包含的并不是对象本身，而是一个指向该对象的指针
  • 从一个变量向另一个变量复制引用类型的值，复制的其实是指针，因此两个变量最终都指向同一个对象
