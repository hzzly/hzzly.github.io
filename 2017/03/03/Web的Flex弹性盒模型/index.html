Web的Flex弹性盒模型

2017.03.03 hzzly

一、Flex布局是什么？

Flex是Flexible Box的缩写，意为”弹性布局”，用来为盒状模型提供最大的灵活性。
任何一个容器都可以指定为Flex布局。

行内元素也可以使用Flex布局。

二、容器的属性

  • [ flex-direction ]
  • [ flex-wrap ]
  • [ flex-flow ]
  • [ justify-content ]
  • [ align-self ]
  • [ align-items ]
  • [ align-content ]
  • [ order ]
